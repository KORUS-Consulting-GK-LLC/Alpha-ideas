# Research

`research/` хранит проверку гипотез, интервью, конкурентный анализ, обзор источников и другие материалы, которые помогают понять, стоит ли развивать идею.

## Правила

- Для каждой идеи использовать папку `research/<slug>/`.
- Основной файл исследования: `research/<slug>/research.md`.
- Начинать с гипотез и вопросов, а не с вывода.
- Отделять факты от предположений.
- Для актуальных внешних фактов указывать источники и дату проверки.
- Завершать research кратким решением: продолжать, уточнить, поставить на паузу, закрыть, вынести в план.
