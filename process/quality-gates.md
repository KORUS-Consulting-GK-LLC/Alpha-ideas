# Quality Gates

Quality gates помогают понять, когда идея готова перейти на следующий этап.

## Gate 1: Из Inbox В Triage

- Есть карточка идеи.
- Сохранен исходный импульс.
- Есть дата, автор, slug и первичные теги.
- Идея добавлена в индекс.

## Gate 2: Из Triage В Enrichment

- Понятна базовая суть.
- Есть предполагаемая аудитория или пользователь.
- Есть хотя бы один возможный источник ценности.
- Сформулированы открытые вопросы.

## Gate 3: Из Enrichment В Research

- Есть критичные гипотезы.
- Понятно, что именно нужно проверить.
- Определены источники: внутренние, внешние, интервью, прототип.

## Gate 4: Из Research В Planning

- Research содержит выводы.
- Факты отделены от предположений.
- Есть решение, зачем делать план.
- Понятен первый проверяемый результат.

## Gate 5: Из Planning В Pilot

- Есть цель, scope и ответственные.
- Есть критерии успеха.
- Есть ограничения и риски.
- Есть список артефактов, которые должны появиться.

## Gate 6: Export

- Есть handoff.
- Указан новый корень проекта или репозиторий.
- В карточке идеи стоит статус `exported`.
- Ссылки на ключевые research, plans и artefacts сохранены.
