# Память проекта

`memory/` хранит долговременную проектную память: устойчивые решения, правила, глоссарий, критерии оценки и ссылки, которые должны переживать отдельные сессии агентов.

## Индекс

- [project.md](project.md) — назначение и устройство проекта.
- [decisions.md](decisions.md) — журнал устойчивых решений.
- [glossary.md](glossary.md) — словарь терминов и статусов.
- [evaluation.md](evaluation.md) — критерии оценки и готовности идей.
- [../LOCALIZATION.example.md](../LOCALIZATION.example.md) — шаблон локальной карты окружения участника.

## Что сюда не складывать

- Текущие планы и временные черновики.
- Локальную историю агента и adaptive score.
- Секреты и чувствительные данные.
- Большие артефакты конкретной идеи.
