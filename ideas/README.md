# Идеи

`ideas/` хранит карточки идей департамента.

## Правила

- Одна идея — один Markdown-файл.
- Имя файла: `YYYY-MM-DD-<slug>.md`.
- `slug` должен совпадать с именами папок в `research/`, `plans/` и `artefacts/`.
- Новая карточка создается по единственному каноническому шаблону [../templates/idea-card.md](../templates/idea-card.md).
- Не хранить отдельные копии шаблона в `ideas/`, чтобы шаблон карточки не расходился с `templates/idea-card.md`.
- Исходный импульс автора сохраняется отдельно от агентного обогащения.
- Статус меняется в карточке идеи, а не через перемещение файла.

## Индекс

Пока идей нет. Новые идеи добавлять сюда вручную или через агента.
