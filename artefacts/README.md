# Артефакты

`artefacts/` хранит материалы, создаваемые вокруг идей: схемы, прототипы, документы, изображения, экспортные файлы, заметки встреч и другие результаты работы.

## Правила

- Для каждой идеи использовать папку `artefacts/<slug>/`.
- Рядом со значимой схемой или документом хранить контекстный Markdown-файл: исходные вводные, допущения, решения.
- Схемы рисовать в формате `.drawio`.
- Артефакты, важные для дальнейшей работы, должны быть связаны ссылками из карточки идеи, research или plan.
- Не хранить временный мусор и приватные секреты.

## Рекомендуемая структура папки идеи

```text
artefacts/<slug>/
  README.md
  context.md
  diagrams/
  prototypes/
  exports/
  meetings/
```
